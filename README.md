# 📈 PropJournals — Trade Journaling Web App

PropJournals is an open-source web application designed to help 
prop traders track, analyze, and improve their trading performance 
through structured journaling.

## 🌟 Features
- Log and track trading entries and exits
- Journal notes for each trade
- Clean and responsive UI
- Built for prop traders and retail traders alike

## 🛠️ Technologies Used
- HTML5 & CSS3
- JavaScript (ES6+)
- React.js

## 🚀 How to Run
1. Clone this repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser

## 🎯 Who Is This For?
- Prop firm traders
- Retail traders
- Anyone who wants to journal and review their trades
